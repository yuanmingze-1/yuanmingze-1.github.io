幽灵化

声明：本游戏非原创,转载自js13kgames.com

幽灵化是一款类似推箱子类型的益智游戏，你扮演一个裸体的外星人类，他需要移动 13 步才能变成幽灵。目标是收集地图上的所有金币，但转折是，只有在你是幽灵的时候才能收集，但作为幽灵，你不能推动石头或按按钮。

游戏有 12 个关卡，还有一个隐藏关卡，通过击败所有其他关卡来解锁。

控制

箭头键 ：移动

R ：重新启动关卡

删除或 Z ：撤销一步

回车或 Esc ：打开暂停菜单

!!! 重要 !!!

1.使用 Firefox 或 Chrome/Edge 来确保音频正常工作。显然，Brave 浏览器无法正常工作，而且很可能 Safari 也无法正常工作。

2.图形在 Firefox 的私密模式下播放时显示不正确（可能会出现一些噪点）。在常规模式下运行正常。原因不明。
